# Protocol: Meeting Prep

## Triggers
"prep for my [meeting/event]", "what's on my calendar at [time]", "help me prep for [event]"

## Steps
1. Get the calendar event from Google Calendar (subject, time, attendees, location, description).
2. Search Gmail for related threads:
   - By subject keywords
   - From/to attendees
   - In the past 30 days
3. Search Google Drive for related docs (subject keywords + attendee names).
4. Check `knowledge/` for any vendor/contact/topic file matching the meeting subject.
5. Check `memory/` for prior context.

## Output
- **Meeting:** subject, time, attendees, location
- **Context:** what this is about (1-2 sentences)
- **Recent thread:** key points from latest email exchange
- **Relevant docs:** Drive files with brief summaries
- **Prep questions:** what to think about beforehand
- **Suggested talking points:** if applicable

## After prep
Ask: "Anything specific you want to dig into before the meeting?"
