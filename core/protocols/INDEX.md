# Protocols Index

Operational playbooks that run on-demand or on cadence. When the user invokes a trigger phrase, follow the corresponding protocol exactly.

- [Morning Briefing](morning-briefing.md) — Daily start: calendar, inbox, todos, focus
- [Weekly Review](weekly-review.md) — Sunday: completions, backlog, financial check, lookahead
- [Email Triage](email-triage.md) — Process Gmail inbox, surface what needs attention
- [Meeting Prep](meeting-prep.md) — Gather context before a calendar event
- [Subscription Audit](subscription-audit.md) — Find recurring charges in Gmail
- [Receipt Tracking](receipt-tracking.md) — Extract orders/receipts from Gmail
- [Todo Extraction](todo-extraction.md) — Pull action items from forwarded emails
- [Monthly Financial Check-In](monthly-financial-check-in.md) — Cashflow, accounts, plan progress
- [Dream](dream.md) — Auto: lightweight memory consolidation at session startup when 24h+5sess elapsed (or 72h floor)
- [System Maintenance](system-maintenance.md) — Monthly: prune memory, promote patterns, process inbox, verify indexes
- [Quarterly Self-Assessment](quarterly-self-assessment.md) — Every 3 months: effectiveness, calibration, skill usage, sycophancy check, user grades the assistant
- [Multi-Session Coordination](multi-session-coordination.md) — Session registry, locks for pillar files, append-only conventions for safe concurrent writes
